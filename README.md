# usymmij.xyz 

Personal Website 

> Built with Rust + Leptos, compiled with Trunk

# Compile and serve
using [trunk.rs](https://trunkrs.dev/)

## development
- opens in new tab
- updates with file changes
```bash
trunk serve --open
```


## Build
- produces files for serving
    - build files are in the `dist/` folder
```bash
trunk build
# or
trunk build --release
```
